# Iasi.jl
Read utilities for IASI granules

Developing Read utilities in Julia for EUMETSAT IASI granule files (both uncompressed binary files and hybrid PC principal component compressed hdf5 files)

IASI Level 1 Product Guide: https://www.eumetsat.int/media/44030
IASI Level 1 Product Format Specification: https://www.eumetsat.int/media/38678

IASI PCH Product Data: https://www.eumetsat.int/changes-pc-compressed-iasi-l1c-data
